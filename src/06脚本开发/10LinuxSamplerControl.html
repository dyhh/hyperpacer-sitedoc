Linux性能数据收集器

功能特色

监控负载服务器的运行情况是很重要的。Linux性能数据收集器提供了监控Linux服务器的功能。 它能够帮助测试人员监控平台上的CPU、内存、Swap、磁盘I/O和网络I/O。

配置界面

注意! 鼠标悬停在界面中的任意标签或控件上，可以看到详细描述信息。

参数详解

服务器名称或IP：Web服务器的域名或者IP地址，例如www.example.com[不包括前缀http://]

端口：Web服务器监听的端口号，默认值为22

认证方式：包括密码认证和密钥认证两种

用户名：服务器的用户名

密码：服务器的密码

密钥：私钥

密钥文件：使用公钥加密过的文件

取样间隔(s)：取样间隔时间，默认为1s

取样条数：每次取样的条数，默认为10条

验证连接：检验连接是否成功，验证连接成功后才可以进入“资源监控视图”选择性能指标。

监视指标：

• CPU

• CPU Perc

• Memory

• Disk

• Network
