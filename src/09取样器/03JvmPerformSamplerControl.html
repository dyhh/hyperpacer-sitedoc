JVM资源监控取样器

功能特色

通过该取样器可以监控JVM资源使用情况。

配置界面

注意! 鼠标悬停在界面中的任意标签或控件上，可以看到详细描述信息。

参数详解

服务器名称或IP：Web服务器的域名或者IP地址，例如www.example.com[不包括前缀http://]

端口：Web服务器监听的端口号，默认值为22

用户名：服务器的用户名

密码：服务器的密码

取样间隔(s)：取样间隔时间，默认为1s

验证连接：检验连接是否成功，验证连接成功后才可以进入“资源监控视图”选择性能指标。

监视指标：

(1) Operating System

• Committed Virtual Memory Size

• Free Physical Memory Size

• Free Swap Space Size

• Process Cpu Time

• System Load Average

• Total Physical Momory Size

• Total Started Thread Count

• Total Swap Space Size

(2) Threading

• Current Thread Cpu Time

• Current Thread User Time

• Peak Thread Count

• Thread Count

(3) Memory

• Committed Heap Memory Usage

• Max Heap Memory Usage

• Used Heap Memory Usage

(4) GC:PS Mark Sweep

• Garbage Collector PS Mark Sweep Duration

• Garbage Collector PS Mark Sweep Collection Count

• Garbage Collector PS Mark Sweep Collection Time

(5) GC:PS Scavenge

• Garbage Collector PS Scavenge Duration

• Garbage Collector PS Scavenge Collection Count

• Garbage Collector PS Scavenge Collection Time
